# Directory Size Calculator Script

## Description
This script calculates the total size of all files in a specified directory, including subdirectories, and returns the result in megabytes (MB).

## Requirements
- Python 3.x
- `os`

## Usage
```bash
python3 dir_size_calculator.py
```

## Parameters
- The script prompts the user to input a directory path.

## Example
```bash
python3 dir_size_calculator.py
Specify the directory path: /path/to/directory
```

## Features
- Verifies if the directory exists before calculating.
- Recursively calculates the size of all files in the specified directory.
- Outputs the total size in MB.
